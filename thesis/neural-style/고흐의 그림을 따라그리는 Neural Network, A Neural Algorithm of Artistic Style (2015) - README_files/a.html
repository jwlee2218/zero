                                SangHyuk Chun Mod Thi Vu 3 years ago

                                1. Noise image itself is not important. The point is that, this problem is a kind of optimization problem:

                                x = \arg\min_x L_tot (x, a, p)

                                where x is output of the algorithm (generated image), a and p are content and style image respectively. This algorithm solve this problem using gradient descent (backpropagation) that means, you need gradient dL/dx, or dL/dF. random image x is a just initial point of the algorithm. It is 'completely random' images not related to content image or style image (but you can design such kind of initialization method)
