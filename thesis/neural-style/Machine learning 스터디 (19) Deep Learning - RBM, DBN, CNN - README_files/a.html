                                          안녕하세요
                                          공부를 막 시작하시는 입장이시니 이것저것 궁금한 것들이 많으시겠죠 ㅎㅎ 황당한 질문일것까지는 아닐 것 같습니다.
                                          CNN 이라는 모델 자체는, 일반적인 목적으로 설계된 모델이기 때문에 그 어떤 데이터를 input으로 써도 상관없습니다. 이미지를 input으로 넣는것도 픽셀 데이터를 집어넣는 것이기 때문에 실제 데이터를 까보면 143 231 224 10 ... 이렇게 real number 들이 array 형태로 배열되어있습니다.
                                          다만, CNN의 convolution이나 pooling의 개념은 2D array 형태의 데이터에 적용되는 개념이기 때문에 데이터를 2D 형태로 바꾸거나 (즉, 100 차원 데이터를 10 x 10 array 형태의 데이터로 변형하거나) 아니면 지금과는 조금 다른 형태의, 조금 더 일반적인 데이터들에 적용할 수 있는 CNN 모델이 필요하겠죠
                                          충분한 답변이 되었으면 좋겠습니다. 감사합니다
